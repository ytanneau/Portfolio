Projet : Le Snake autonome
Date : 2024
Contexte : Projet de formation
Technologie : C

Ce projet est la suite du projet Le jeu du Snake.
Le but est d'automatiser le serpent pour finir le jeu, le tout en faisant de moins de déplacement possible. Pour cela je repere le chemin le plus cour parmis le chemin directe ou par les portail. Et je sauvegarde les chemin dans un tableau qui est un calcule du plateau.
